## Getting Started

1. Clone this repository and navigate to MolSculpt folder.

```
git clone https://github.com/SakuraTroyChen/MolSculpt.git
cd MolSculpt
```

2. Following [NeXT-Mol](https://github.com/acharkq/NExT-Mol) to configure your environments.
